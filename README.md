This project is about a social network model to sample a complex graph in order to achieve a simpler graph with the most resemblance to original one using random edge and shortest path between two random nodes methods.In this project, the goal is to achieve a smaller graph with the most similar behaviour to the original one.
HYB algorithm was applied in random edge sampling and shortest path between two random nodes sampling used Dijkstra algorithm.
Sampling is done in scale-down and back-in-time approaches and result of each one was conducted with D-statistic index to evaluate the amount of resemblance. 
This project is done on Google Colab using python and libraries such as networkX, pandas, matplotlib.pyplot and Little ball of fur are used.  
In this project dataset is a social network of LastFM users. 
For each sampling method centrality and diffusion models such as Independent Cascade and Linear Threshold were assessed and two sampled graphs were compered.
